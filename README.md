# Gloss and Git

A Vite + React + TypeScript landing page for Gloss and Git, a curated women in tech and AI community in Seattle.

## Local Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
